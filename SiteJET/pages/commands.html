Introdução

Bot desenvolvido pra atender usuários do servidor de dicord podendo atender diversas áreas como lojas no próprio Discord

Sobre o bot

Bot desenvolvido pra um projeto da universidade no segundo semestre do ano de 2022

Tecnologias

Bot Desenvolvido utilizando a biblioteca do Discord.js

Utilizamos o próprio discord como banco de dados pra armazenar os tickets fechados e até mesmo as próprias imagens e conteúdo utilizado pelo BOT.

Começando

Para começar com o nosso bot é bastante fácil e fizemos o nosso melhor para torná-lo mais simples para vocês.

Noções básicas

Aqui no site temos as listas de todos comandos disponíveis.
Você também tem a opção de obter ajuda em todos os comandos do bot diretamente através do discord com o uso do comando de ajuda do bot.

Importante: Para utilizar o bot você deve estar ciente dos termos de uso .
Use /termos para mais informações.

Adicionando o BOT

Para adicionar o bot no seu Servidor! clique aqui.

Você não consegue coloca-lo em servidores no qual você não tem permissão de administrador.
